Kaynak Kodundan Nasıl Derlenir veya Direkt Olarak Çalıştırılır?

	180202024-180202025 adlı arşivi çıkardıktan sonra aynı isimli klasör içerisinde projenin
kaynak kodları bulunmaktadır.İsterseniz maven projesi olarak destekleyen herhangi bir java idesinde
klasörü importlayarak projeyi kaynak kodu üzerinden derleyebilir ve çalıştırabilirsiniz.
	
	Direkt çalıştırmak için ise yine aynı klasör içerisindeki MultiThreadAsansorSistemi.jar
isimli çalıştırılabilir jar dosyasından direkt olarak çalıştırabilirsiniz.(Sisteminizde java
doğru bir konfigürasyonla kurulu olmalıdır.)

Nasıl Kullanılır?
	
	Program sadece çıktı veren bir form uygulamasıdır. Sistemi başlat butonuna bastıktan sonra
ilgili çıktıyı dinamik arayüz üzerinden takip edebilirsiniz. Arayüz etkileşimli değildir. Sadece
sistemi izlemeniz içindir.
